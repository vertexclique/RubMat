Installing

$ gem install RubMat

after that if you want to fall into command line of rubmat, type:

$ rubmat

or if you want to use librbdiscrete provided by RubMat get in your rubygems's gem directory and load modules via irb from folder lib/

Replace first command in MacOS with;

$ sudo gem install RubMat
